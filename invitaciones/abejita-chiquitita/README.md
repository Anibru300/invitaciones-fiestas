# 🐝 Invitación: Abejita Chiquitita

Plantilla de invitación con temática de **Maya la Abeja / La abejita Chiquitita**.

## Cómo personalizar

1. Abre `datos.md` y reemplaza los valores ficticios por los datos reales del evento.
2. Edita `index.html` con:
   - Nombre del festejado/a.
   - Edad.
   - Fecha, hora y lugar.
   - Nombre de los papás.
   - Número de WhatsApp para confirmar.
3. Abre `index.html` en tu navegador para ver el resultado.

## Vista previa

Para ver la invitación, simplemente haz doble clic en `index.html` o abre una terminal en esta carpeta y ejecuta:

```bash
# Opcional: levantar un servidor local
python -m http.server 8000
```

Luego abre `http://localhost:8000` en tu navegador.
